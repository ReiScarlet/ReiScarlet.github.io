网络设备

Posted on 2021-04-02 | In 计算机网络

网络设备及部件是连接到网络中的物理实体。网络设备的种类繁多，且与日俱增。基本的网络设备有：计算机（无论其为个人电脑或服务器）、集线器、交换机、网桥、路由器、网关、网络接口卡（NIC）、无线接入点（WAP）、打印机和调制解调器、光纤收发器、光缆等

1.集线器

  1. 目的：解决终端主机的物理连接问题，将在其内部所有的网络设备联通
  2. 不过滤任何数据，也不知道数据发送到什么地方
  3. 终端集中点，通过集线器形成星型拓扑
  4. 物理层设备
  5. 对信号整型放大
  6. 共享一个信道，性能比较差
  7. 会将信息传到除自己以外的所有主机，哪怕只想和其中的一个通信，造成浪费带宽
  8. 因为不能读取IP地址，所以不能在外部网络（例如互联网）交换数据

2.交换器

  1. 数据链路层设备，能够解析数据帧结构
  2. 使用MAC地址，能够根据MAC地址进行主机的定位寻址，所以信息只会发送到目的主机减少带宽的浪费和集线器不同
  3. 独享信道，所以交换机的性能远远超过集线器
  4. 不能读取IP地址

3.路由器

  1. 实现网络连接进行数据交换
  2. 网络层设备
  3. 可以读取IP地址，所以可以将数据包发送到外部网络
  4. 输入输出端口安装了三种协议软件，分别是物理层，数据链路层，网络层

总之，集线器和交换机构建网络，路由器将网络与网络连接起来

4.防火墙

防火墙是一个系统，用来防止未经授权的网络传输侵入私有的网络系统，是由过滤传输的数据来达到这个效果

  1. 目的：要在公共网络和私人的网络间建立一道流量管控机制
  2. 防火墙由网络管理员管理和制定规则
  3. 防火墙可以针对网域名，协议，软件，IP地址，端口和关键词来制定规则
  4. 防火墙有多种形式，一种是主机式防火墙，一种是软件式防火墙，网络式防火墙

5.无线接入点（WiFi）

  1. 延申网络覆盖范围
  2. 移动设备接入点
