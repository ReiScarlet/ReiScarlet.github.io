向量空间与子空间

2021-04-11

向量空间, 读书笔记

向量空间

一个向量空间是有一些被称为向量的对象构成的非空集合V，在这个集合商定义了两个运算，加法和标量乘法，服从以下公理，这些公理必须对V中所有向量u，v，w及其所有标量c和d成立

  1. u+v属于V
  2. u+v=v+u
  3. (u+v)+w = u + (v + w)
  4. 存在一个零向量0， u + 0 = u
  5. V中存在一个向量-u，u + (-u) = 0
  6. cu属于V
  7. c(u+v) = cu + cv
  8. (c+d)u = cu + du
  9. c(du) = (cd)u
  10. 1u = u

子空间

定义：向量空间V的一个子空间是一个满足以下三个性质的子集H：

  1. V中零向量属于H
  2. 对H中的任意向量u和v，向量u+v属于H
  3. 对H中的任意向量u和c，向量cu属于H

定理1：若$v_1,…v_p$在向量空间V中，则$span\{v_1,…,v_p\}$是V的一个子空间

生成(张成)子空间：$Span\{v_1,…,v_p\}$是由$\{v1,…,v_p\}$生成(张成)

生成(张成)集：给定一个子空间H，H的生成集是集合$\{v1,…,v_p\} \subset H$,满足$H = span\{v1,…,v_p\}$

矩阵的零空间

定义：矩阵A的零空间写程Nul A，是齐次方程Ax=0的解集，用集合符号表示即是$Nul\ A = \{x:x\in \mathbb{R^n}, Ax=0\}$

定理2：m*n的矩阵A的零空间是$R^n$的一个子空间，等价地，m个方程、n个未知数的齐次线性方程Ax=0的全体解的集合是$R^n$的一个子空间

注意：

  1. Nul A包含非零向量时，它的生成集中向量的个数等于方程Ax=0中自由变量的个数
  2. Nul A的生成集，自由变量是生成向量上的权时，生成集必然是线性无关的，例如$x_2\begin{bmatrix} 2 \\ 1 \\ 0 \\ 0 \\ 0 \end{bmatrix} + x_4\begin{bmatrix} 1 \\ 0 \\ -2 \\ 1 \\ 0\end{bmatrix} + x_5\begin{bmatrix} -3 \\ 0 \\ 2 \\ 0 \\ 1\end{bmatrix} = x_2u + x_4v + x_5w$，$x_2,x_4,x_5$就是自由变量，$u,v,w$就是生成集，所以它们线性无关

矩阵的列空间

定义：m*n矩阵A的列空间是由A的列的所有线性组合组成的集合，若$A=[a_1,…,a_n]$,则$Col\ A = Span\{a_1,…,a_n\}$

定理3：m*n矩阵A的列空间是$\mathbb{R^m}$的一个子空间

因为$Col\ A$是A所有列的线性组合，则Ax可表示A的列向量的一个线性组合，所以$Col\ A = \{b:b=Ax, x \in \mathbb{R^n}\}$，这代表$Col\ A$中向量记号Ax也表明$Col\ A$是线性变换$x\rightarrow Ax$的值域

m*n矩阵A的列空间等于$\mathbb{R^m}$当且仅当方程$Ax=b$对$\mathbb{R^m}$中每个b都有一个解

线性变换的核与值域

定义：由向量空间V映射到向量空间W内的线性变换T是一个规则，它将V中每个向量x映射成W中唯一向量T(x)，且满足：

  1. T(u + v) = T(u) + T(v)，对V中所有u，v成立
  2. T(cu) = cT(u)，对V中所有u和数c成立

线性变换T的核（或零空间）是V中所有满足T(u)=0的向量u的集合

T的值域是W中所有具有形式T(x)的向量的集合
